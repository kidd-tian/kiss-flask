kiss-flask
==========

flask project demo


## Guide

* require greater than `python 2.7.6`

### Init environment

```
script/bootstrap
```

### Install ve

`ve` is a tool like `bundle` in `ruby on rails`

see [here](https://github.com/baijian/ve) to install `ve`

### Install python module

```
ve pip install Flask-Cache
```

### Run it

```
ve python run.py
```
